# The Real Business Problem (Not Just “Load CSV Files”)
In many retail systems, data is not always final when it first arrives.

During this project, I worked on a real-world scenario involving a large retail company that collects daily Point-of-Sales (POS) transactions from multiple store locations. These transactions arrive as daily CSV files in Azure Data Lake Storage (ADLS).
